小天盒子开源许可

小天盒子参考项目如下，感谢以下开源项目。

小天盒子V5.0.0之后有极大可能不再维护，有想加入小天开发或者构建分支的请查看 https://github.com/surine/TustBox

小天盒子暂只有Android版，如果有意开发微信小程序或者ios版本请关注后期开放的API，（由于微信小程序的强制https及ios上架AppStore的登录验证可能直接实现非常困难，所以可以先通过Android端同步云课表，再在小程序或者iosApp上显示），具体可以私聊我。

  • Eventbus

    Event bus for Android and Java that simplifies communication between Activities, Fragments, Threads, Services, etc. Less code, better quality.

  • Okhttp

    An HTTP & HTTP/2 client for Android and Java applications

  • Jsoup

    jsoup: Java HTML Parser, with best of DOM, CSS, and jquery

  • Litepal

    An Android library that makes developers use SQLite database extremely easy.

  • Circleimageview

    A circular ImageView for Android

  • Glide

    An image loading and caching library for Android focused on smooth scrolling

  • ScratchView

    ScratchView repo is UX Design involving scratch cards like views which are scratched to reveal the information they conceal. There are two types of ScratchView

  • Gson

    A Java serialization/deserialization library to convert Java Objects into JSON and back

  • BaseRecyclerViewAdapterHelper

    BRVAH:Powerful and flexible RecyclerAdapter

  • Butterknife

    Bind Android views and callbacks to fields and methods.

  • Ninegridview

    类似QQ空间，微信朋友圈，微博主页等，展示图片的九宫格控件

  • Badgeview

    支持自由定制外观、拖拽消除的MaterialDesign风格Android BadgeView

  • Aria

    下载可以很简单

  • GsyVideoPlayer

    视频播放器（IJKplayer、ExoPlayer、MediaPlayer）

  • Phoenix

    The one-stop solution for taking pictures / videos, picture / video selection, editing and compression on the Android platform.

  • LabelsView

    Android的标签列表控件。可以设置标签的选中效果。 可以设置标签的选中类型：不可选中、单选、限数量多选和不限数量多选等， 并支持设置必选项等功能。

  • photoview

    Implementation of ImageView for Android that supports zooming, by various touch gestures.
